Troubleshooting

  • Share the file: Right-click the file in Google Drive → Share → Change to "Anyone with the link" → Copy link
  • URL format: Should be https://drive.google.com/file/d/FILE_ID/view
  • Check console: Open DevTools (F12) → Console tab to see conversion logs
  • Test direct URL: If preview fails, try opening the converted URL directly in a new tab
